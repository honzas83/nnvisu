# Quickstart: GUI Dynamic Classes

## New Features

1.  **Color Palette**: Select from 8 high-contrast colors to add points for different classes.
2.  **Eraser**: Select the eraser tool to delete points by clicking or dragging.
3.  **Play/Pause**: Use the single toggle button to control training.
4.  **History Seekbar**: Scrub through training history when paused.
5.  **Dynamic Architecture**: Add/remove classes on the fly; the model adapts weights incrementally.

## Verification Steps

1.  **Dynamic Classes**:
    - Select 3 different colors and add points.
    - Start training and observe 3 regions.
    - Add a 4th color and see the model adapt.
2.  **Eraser**:
    - Select the Eraser.
    - Click on a point to remove it.
    - Drag over multiple points to erase them.
3.  **History**:
    - Train for ~50 steps.
    - Click Pause.
    - Drag the seekbar to see previous decision maps.
4.  **Branding**:
